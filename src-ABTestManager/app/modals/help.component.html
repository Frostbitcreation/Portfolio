A/B Test Manager

Help

Introduction

This tool allows you to view and to configure individual experiments, altering the percentage of users assigned to each variation of an experiment, as well as the duration of an experiment.
Each Google Optimize container can have up to 5 tests active at any time, with the cookie duration being up to 90 days.
New tests can be made in Google Optimize but will not show in the tool unless it's details are stored internally, which will then allow for lmited chnages to be made to the experiment after it is running.

✔ You'll use the tool to manage the duration a test is running for.
✔ You'll use the tool to make changes to variations.
✕ You won't use the tool to make new experiments.

How Experiments Work

An experiment is set up within Google Optimize and connected to a Google Analytics account that collates the desired data from the experiment.
When a user accesses a site with a test, a check is done for a test cookie within the user data.
If no cookie is found matching the name/values of the active site experiment, the user is assigned an experiment and their experience continues unimpeded.

Creating Experiments

A selected container, or in our case fascia, can have a maximum of 5 active experiments.
You can schedule experiments to start and end at predetermined times within google optimize but the internal records of these experiments need to be updated to match.
As a result, experiments are preferred to be started and ended manually.


Variants

An experiment can have up to 8 variations and assigned a weight value.
This percentage value is the chance of a variation to be assigned to a user that is a part of the experiment.
The values of the experiments must add up to 100% to allow the experiment to be valid.


Editing

Aspects of the experiments such as the weight of each variant can be adjusted while a test is live
Every other value is recommended to be left unchanaged during a live experiment and are unavailable to be changed in the tool as a result.
If any changes are needed or wanted, including changing the weights of variants in an experiment, it is recommended that a new experiment be made that is a copy using the new values.
