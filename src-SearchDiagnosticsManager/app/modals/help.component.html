Search Diagnostic Manager

Help

Introduction

Search Diagnostics Manager is a collection of tools that will allow you to make a search against a live index (product search) or see the stats for past searches (search stats).

✔You will use the tool to search with a product code against an index.
✔You will use the tool to view statistical data from past searches.
✕You will not use the tool to fix product data or search index configurations.

How Search Diagnostics Manager Works

The search diagnostics tool is powered by the live indices in Elasticsearch.

At the bottom of each page the section labelled Raw Search Data is a collapsible section that shows the entire search result from your search.
Discrepancies between returned data that is not shown in the selected result can be identified here.

Product Search

The product search tab is built around the product indices that contains product search data, an index.
The product search page is built to allow searches against the product indices, indices may contain the search data for one or more websites;
indices staring with "sd-search" are related to all Sports Direct websites and apps, an index such as "lifestyle-search" can contain data for multiple fascias.

The tool will not allow a search until an index has been selected to search against.
After selecting an index, entering a valid product Id (2 letters or numbers followed by 4 numbers e.g. AL4894 or 781108) will allow you to submit a search on the index.

When data is returned, the tool will select the first available website (Alphabetical order A-Z) and colour code (Numerical order descending) and displays that product's data.
Changing the selected website will automatically select the first available colour, updating the shown product data.
Changing the selected colour will update the selected product data.

Search Stats

The search stats page is built to find search statistics for a given search, when making a search the returned result is based on any searches within the last 7 days that exactly match the given search term.

Upon a successful search the product data shown will default to an available fascia.
Selecting a new website will show the most recent search data for that website.
In the event that there is no search data across any webpage, going to a website and making the search will (after a few minutes) update the search stats index with the new search.
